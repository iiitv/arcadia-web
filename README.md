## Running Locally
#### Clone the project
  ```sh
  $ git clone https://github.com/iiitv/arcadia-web.git
  ```
---
#### Install the requirements
  ```sh
  $ cd arcadia-web
  $ sudo -H pip install -r requirements.txt
  ```
---
#### Run the server
  ```sh
  $ cd arcadia-web/arc
  $ python manage.py runserver
  ```
